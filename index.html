Bookmarklet Code

These bookmarklets are created with pure JavaScript and can be used on any website. Here's a simple example of how a bookmarklet works:

javascript:(function() {
  // Your JavaScript code goes here
  alert('Hello from a bookmarklet!');
})();

Each bookmarklet follows this pattern but with more complex functionality.

About This Project

This collection of bookmarklets was created for Hack Club's "Hacklet" You Ship, We Ship challenge. The goal was to create fun, useful, or cursed bookmarklets that deserve their own domain.

What's a Bookmarklet?

A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands. When clicked, it performs some function on the current webpage instead of navigating to a new page.

Bookmarklets are a lightweight and accessible way to enhance your browsing experience without installing extensions.
